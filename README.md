# Aastha Rajput — Personal Portfolio & Design System

A bespoke, high-end editorial digital showcase and interactive design catalog presenting selected works in Digital Product Interfaces, Growth Performance Marketing Campaigns, Packaging Blueprint Layouts, and Social Motion Creatives.

---

## ⚡ Design Architecture & Philosophy

The interface is constructed around a clean, structural layout grid reflecting physical design journals. 
* **Cinematic Color Space**: Set in high-contrast dark space (`#0a0a0a`) paired with warm crème paper text elements (`#f2ead9`) and terracotta red highlights (`#c94a4a`).
* **Grid Order**: Features a sticky sidebar menu index on desktop and a highly responsive modular project card grid.
* **Micro-interactions**: Incorporates delayed ring custom cursor follows, smooth scroll reveals, and video previews that autoplay muted on hover or viewport entry.

---

## 📁 Repository Showcase Structure

The project resources are organized into structured directories showing diverse design deliverables:
* `/website design`: Interface recordings and Figma screenshots of boutique web experiences (Maison, Bloom Studio, Motion, Obscur-I).
* `/ad creatives`: Curated performance marketing banners and graphic vector assets.
* `/email and landing page design`: High-converting modular storefront screenshots and welcome sequence newsletters.
* `/packaging design` & `/manual design`: Technical packaging box mockups, Instruction manuals, and print specifications.
* `/videos` & `/reel covers`: High-tempo social reel campaigns and cover layouts.

---

## 📊 Savyo Growth Case Study Dashboard

Includes an embedded campaign telemetry dashboard showcasing performance marketing data:
* **Metrics Tracked**: Total Purchases, Spend, Revenue, Net Profit, and ROAS.
* **Interactive Breakdown**: Features an interactive row selector that updates detail cards dynamically with calculations for product-specific revenue share.

---

## 🛠️ Tech Stack & Implementation

* **Base**: Pure Semantic HTML5 & Vanilla CSS3 Custom Properties.
* **Grid**: CSS Flexbox and fractional 12-column grid tracks.
* **Logic**: Vanilla ES6 JavaScript (No frameworks, lightweight loading).
* **Observer APIS**: IntersectionObserver loops for video playback management and content reveals.
